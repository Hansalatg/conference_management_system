# Web-Based Conference Management System

## Overview

This project is a web-based conference management system developed to streamline conference activities efficiently. The system allows admins and participants to manage conference-related activities through a secure and responsive platform.

## Features

* Participant Registration and Login System
* Admin and Participant Management
* Conference Schedule and Track Management
* Proceedings Upload and Sharing
* QR Code-Based Session Attendance Tracking
* Email Notification System
* Admin Dashboard and Analytics
* User Management
* File Upload Handling
* Responsive User Interface

## Technology Stack

* HTML
* CSS
* JavaScript
* Bootstrap 5
* Node.js
* Express.js
* MySQL

## Installation

1. Clone the repository:
git clone <repository-link>

2. Navigate to the project folder:
cd conference-management-system

3. Install dependencies:
npm install

4. Configure the MySQL database and import the SQL file.

5. Start the server:
npm start

6. Open the application in the browser:
http://localhost:3000
